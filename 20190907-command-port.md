# Run Program & Command Line & Default Argument

### Run Program
  - python foo.py
  - python -m foo
  
  
### Run Program vs Import Module
  - import foo
  - __name__

### Command Line Argument
  - sys.argv
  - python foo.py ...
  - python -m foo ...

### Default Argument
  - python -m http.server PORT
  - python -m http.server # 8000

### Different Meaning
  - python -m http.server PORT
  - python -m venv ENV_DIR
