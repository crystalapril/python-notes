### 对象概述 outline

    一切皆对象 everything is object
    
    python有句话叫一切皆对象
    什么意思呢，就是你之前学过的，接触过的，都是对象
    
    举个栗子，第一篇里的整数 int 是 object，字符串 string 也是 object，同样 list 也是object 
    包括前天提到的，open 函数打开文档，返回的也是 file object
